## 结论范围

随机背景是 **PU（positive-unlabeled）未标注背景**，不是确认阴性；严格 AUC 只使用留出确认正例与留出确认阴性。靶点层级：{"limited_positive": 658, "strict_labeled": 167, "unlabeled_fallback": 175}。
