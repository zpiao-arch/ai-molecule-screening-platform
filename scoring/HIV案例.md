## 测试设计

**靶点**: HIV-1蛋白酶 (UniProt P03367)

**候选集**: 30个分子 (13个HIV药物 + 17个诱饵)

**评分模式**: 完整四层漏斗


### 评分公式

```
final = L1×0.20 + L2×0.50 + L3×0.20 + L4×0.10
```

| 层 | 权重 | 方法 | 作用 |
|:---:|:---:|------|------|
| L2 | **50%** | DeepPurpose BindingDB CNN_CNN | 淘汰"不结合靶点"的分子 |
| L1 | 20% | RDKit QED/SA/Lipinski/MW/logP/TPSA/HBD/HBA | 淘汰"不像药"的分子 |
| L3 | 20% | 39条SMARTS毒性规则 + DeepChem RF (Tox21/BBBP/ClinTox/SIDER) | 淘汰"可能有毒"的分子 |
| L4 | 10% | Uni-Mol + 1112个FDA药物Top-10相似度 | 偏好"像已知好药"的分子 |

### 候选分子构成

| 类别 | 数量 | 示例 |
|------|:---:|------|
| HIV相关药物 | 13 | 达芦那韦、多替拉韦、齐多夫定、奈韦拉平、马拉韦罗、恩曲他滨类似物等 |
| 常用药物(非HIV) | 4 | 阿司匹林、布洛芬、对乙酰氨基酚、咖啡因 |
| 简单分子/诱饵 | 8 | 苯、萘、PABA、磺胺等 |
| 有毒分子 | 1 | 沙利度胺 |
| 结构类似物 | 4 | 依法韦仑变体、苯甲酸甲酯衍生物等 |

## 运行命令

```bash
python scoring.py \
  -i data/hiv_candidates.csv \
  -o outputs/hiv_result.csv
```

## 各层分数分布

| 层 | HIV均值 | 诱饵均值 | 分离度 | 区间 | 作用 |
|:---:|:---:|:---:|:---:|:---:|------|
| **L2** | **0.610** | 0.323 | **+0.287** | 0.96 | **核心区分维度**，50%权重 |
| L1 | 0.820 | 0.886 | -0.065 | 0.67 | 类药性，简单诱饵天然略高 |
| L3 | 0.679 | 0.714 | -0.035 | 0.33 | 安全性，简单分子略安全 |
| L4 | 0.949 | 0.948 | +0.001 | 0.09 | 通用药物相似度，近无区分 |
| **Final** | **0.700** | **0.576** | **+0.123** | 0.51 | L2权重主导最终排名 |

### 解读

- **L2是核心引擎**：HIV药物均分0.61 vs 诱饵0.32，分离度0.29。在50%权重下决定性地拉开排名
- **L1偏袒简单诱饵**（0.89 vs 0.82）：布洛芬、阿司匹林本身就"非常像药"，L1给他们高分是正确的——它们确实是好药，只是不治HIV
- **L3略微偏袒简单诱饵**（0.71 vs 0.68）：分子越简单越安全。但严格毒性过滤器（三级惩罚）对沙利度胺等有毒分子有明显压制（L3=0.500天花板）
- **L4近无区分**（0.949 vs 0.948）：所有类药分子都高度相似于某些FDA药物

## 技术参数

| 项目 | 参数 |
|------|------|
| L2引擎 | DeepPurpose CNN_CNN |
| L2模型 | BindingDB预训练 (92万条蛋白-配体亲和力) |
| 靶点蛋白 | HIV-1蛋白酶 (UniProt P03367, 99aa) |
| L1参数 | MW甜区150-600, logP甜区-0.5~4, TPSA 20-140, HBA 1-10 |
| L3 SIDER阈值 | 0.8 (仅高置信度副作用) |
| L3 SMARTS | 39条精修毒性基团规则（4大类） |
| L3 惩罚机制 | 红牌(严重基团)→L3≤0.35 / 黄牌(≥2中等)→L3≤0.50 / 重金属→L3≤0.25 |

| L4嵌入 | Uni-Mol预训练, 15层Transformer, 512维 |
