# 高通量测试闭环 (HTS Closed-Loop) 结果报告

> 生成时间: 2026-07-14 | 脚本: `hts_closed_loop.py` | 结果: `hts_closed_loop_results.json`

## 1. 实验设计（真实闭环，非小集 AUC）
- **候选库**: 24,268 个真实药物分子（全库灌入）
- **靶点面板**: 164 个（靶点文本经 chembl_id 精确解析，100% 命中）
- **已知活性基准**: 面板 15,773 个已知活性分子 **100% 落在 24k 库内**（canon 命中），故闭环 = "从真实药库里把已知活性分子筛出来"，零注入
- **打分**: L1(0.20) + L2_BindingDB 合并模型(0.50) + L3_ADMET(0.20)；Uni-Mol(0.10) 为吞吐省略
- **度量**: 全库排序后 recall@K、EF@5%、BEDROC、全库 AUC、级联剪枝；吞吐 = 配对/秒

## 2. 吞吐（✅ 达标）
| 指标 | 值 |
|---|---|
| 配对总数 | 3,979,952 (24k × 164) |
| 墙钟 | 870 秒（14.5 分钟） |
| **吞吐** | **4,574 pair/s** |

L1/L3 为靶点无关、预计算一次缓存；逐靶点仅增量 L2 MLP（向量化），故规模化成本极低。

## 3. 效能（⚠️ 真实且有异质性，必须如实看）
| 指标 | median | 解读 |
|---|---|---|
| 全库 AUC | **0.673** | 真实判别信号（>0.5），但非强 |
| AUC>0.6 靶点占比 | **66%** (108/164) | 多数靶点有可用信号 |
| AUC>0.7 靶点占比 | **40%** (65/164) | 四成靶点较强 |
| EF@5%（富集因子） | **0.852** | 中位数 <1 → 多数靶点 top5% 未富集 |
| EF@5%>1 靶点占比 | **46%** (75/164) | 近半数有真实富集 |
| EF@5%>2 靶点数 | **56** | 强富集靶点 |
| recall@5%（随机基线=0.05） | **0.047** | 中位 ≈ 随机，未集中到顶端 |
| recall@5%>0.2 靶点数 | **30** | 检索可用的靶点 |

**关键结论**：合并 L2 模型在全库排序上有**真实但异质**的判别力（CA2 类靶点 AUC 0.93、EF 9–12、recall@5% 0.58），但**中位数层面未把活性分子集中到排名顶端**（recall@5%≈随机、EF@5% 中位 0.85）。这与早前"小集+decoy 下 90.9% 靶点 AUC>0.5"的乐观结论形成对照——全库虚拟筛选是更严苛的检验。

**典型靶点**：
- ✅ CHEMBL205 (Carbonic anhydrase 2): AUC=0.927, EF@5%=8.97, recall@5%=0.577（397 个活性中 58% 落入 top5%）
- ✅ 最佳: CHEMBL4430/3890/3426: AUC 0.93–0.94, EF 10–12
- ❌ 最差: CHEMBL2414/1836/4611: AUC 0.21–0.33（模型反向，多为数据稀疏/冷靶点）

## 4. 级联剪枝（漏斗算力账）
| 阶段 | 保留分子比例 | 活性保留比例 |
|---|---|---|
| L1 质量门 (≥0.45) | 93.8% | 100% |
| L2 top10% | — | **8.5%**（低于随机 10% → L2 排序未集中活性）|
| L3 ADMET 门 (≥0.5) | 91.4% | 99.3% |

L1/L3 门几乎不剪枝（保留 ~92–94%），L2 top10% 反而只留住 8.5% 活性——说明当前 L2 在"取-top"模式下不优于随机，漏斗的算力节省主要来自后续对接而非 L2 预筛。

## 5. ⚠️ 必须说明的度量陷阱
- **BEDROC median=1.0 是假象**：alpha=20 下，当某靶点活性占比极小（npos/n≈0.004），归一化分母 ra_max 极小，只要活性落在靠前若干位即 ra/ra_max≫1 被截到 1.0。故 BEDROC 在此稀疏场景下已饱和，**不能读作"完美早期富集"**，应以 EF@5% / recall@K 为准。
- 全库 AUC 与 recall@5% 背离是常态：AUC 衡量整体排序优劣，recall@5% 要求活性"扎堆顶端"；后者对药物重定位类任务才是金标准。

## 6. 下一步（提升真实检索效能的方向）
1. **L2 特征升级**：现有 Morgan(512)+文本哈希(256) 对"靶点特异性活性"刻画有限；引入蛋白序列/结合口袋表征或对接打分为 L2 提供结构信号。
2. **重标注活性集**：面板 pos 多为多靶点广谱药，需按靶点的"选择性强活性"重新界定正样本，减少标签噪声。
3. **顶端重排**：对 L2 top 候选用 Boltz-2/MCCE 柔性对接精排（上游已接，待 GPU+einops），把 AUC 信号转化为 top-K 富集。
4. **度量切换**：以 EF@5% / recall@K 为主指标（非小集 AUC）驱动迭代。
